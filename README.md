# Unimed API 🏥

## 📌 Descrição
Projeto de estudo desenvolvido em **.NET/C#** para criação de uma **Web API**.  
O objetivo foi praticar conceitos de desenvolvimento de APIs e integração com banco de dados SQL Server.

## 🛠️ Tecnologias Utilizadas
- .NET / C#
- ASP.NET Web API
- SQL Server
- Visual Studio

## 🚀 Funcionalidades
- Endpoints básicos para manipulação de dados.
- Integração com banco de dados via script SQL.
- Estrutura simples para aprendizado de APIs REST.

## 🎯 Objetivo do Projeto
Explorar conceitos fundamentais de criação de APIs em .NET, incluindo:
- Estruturação de controllers.
- Conexão com banco de dados.
- Manipulação de dados via endpoints.

## 📂 Estrutura do Projeto
- **UnimedWebApi** → Projeto principal da API.
- **SQLQuery1.sql** → Script de banco de dados.
- **.vs** → Configuração do Visual Studio.

## 📖 Aprendizados
- Primeiros passos com ASP.NET Web API.
- Integração de API com SQL Server.
- Estruturação de endpoints REST.
